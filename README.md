## このプロジェクトについて
このプロジェクトは、OAuthの同意疲れを疑似的に体験するためのサンプルアプリケーションです。  
ユーザーが複数の外部サービスに対してOAuthの同意を求められることで、同意を面倒に感じたり、無意識に同意してしまう現象を体験できます。  
[サンプルアプリ](https://sample-many-const-app.maronn.workers.dev/)を作成しているので、動作確認が可能です。  
サンプルアプリはテキストエリアに入力された内容をそのまま送信してください。  

このアプリは[こちらの記事](https://zenn.dev/maronn/articles/about-cross-app-access)で紹介しているOAuth同意疲れを実際に体験してもらうために作成されました。
